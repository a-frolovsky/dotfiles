# D O T F I L E S
This is a collection of my dotfiles for MacOSX.

## Setup
* Run `softwareupdate --install --all`
* Install [homebrew](https://brew.sh/)
* Clone this repo
* `brew_installs.sh`
* `cask_installs.sh`
  * Install other apps from AppStore (XCode, etc)
* `install_oh-my-zsh.sh`
* `install.sh`
* `install_nvim.sh`
